# 🐧 Linux Studies

---

## 📌 Sobre o repositório

Este repositório foi criado para documentar minha evolução no aprendizado de Linux de forma organizada e prática.
A proposta é ir além da teoria, aplicando os conceitos em exercícios reais, pequenos scripts e projetos que simulam cenários do dia a dia.

### Objetivos principais

* Consolidar fundamentos de Linux e Bash.
* Praticar comandos essenciais do terminal.
* Desenvolver scripts para automação de tarefas.
* Aprender ferramentas clássicas do ecossistema Unix.
* Criar uma base técnica consistente para estudos futuros em DevOps, SRE e Engenharia de Software.

## 📚 Tópicos estudados

### Comandos básicos

Estudo dos comandos fundamentais para navegação, manipulação de arquivos e interação com o sistema:

* `ls`
* `cd`
* `pwd`
* `echo`
* `cat`
* `cp`
* `mv`
* `rm`
* `touch`
* `mkdir`

### Processamento de texto

Ferramentas voltadas para filtragem, transformação e análise de conteúdo em arquivos e saídas de comandos:

* `grep`
* `awk`
* `sed`
* `cut`
* `sort`
* `head`
* `tail`

### Shell Script

Conceitos e práticas para criação de scripts em Bash:

* Variáveis
* Arrays
* Operadores `[[ ]]` e `(( ))`
* Estruturas condicionais (`if`, `case`)
* Laços (`for`, `while`)
* Funções
* Argumentos posicionais (`$1`, `$2`, `$@`)
* Redirecionamentos
* Pipes e encadeamento de comandos
* Tratamento básico de erros

### Outros tópicos

Conteúdos complementares importantes para uso mais avançado do sistema:

* Permissões e usuários
* Processos e gerenciamento de tarefas
* Expressões regulares
* Manipulação de arquivos e diretórios
* Logs e auditoria
* Variáveis de ambiente
* Estrutura do sistema de arquivos
* Gerenciamento básico de pacotes

## 🎯 Metodologia de estudo

Meu processo de aprendizado é baseado em prática contínua e documentação. A abordagem inclui:

1. estudar um conceito ou comando;
2. testar no terminal;
3. criar exercícios próprios;
4. transformar a prática em script quando possível;
5. registrar observações e aprendizados;
6. revisar periodicamente para reforçar a retenção.

Essa metodologia ajuda a transformar conhecimento teórico em habilidade prática.
